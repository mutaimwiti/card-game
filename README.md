# card-game
♠️ Card game card ♣️

### Setup
Ensure you have node installed. Use either `npm` or `yarn`.

Run this command to install project dependencies.
```bash
yarn install
```
or
```bash
npm install
```

### Run
Run this command to start the game.
```bash
yarn start:dev
```
or
```bash
npm run start:dev
```

### Run tests
Run this command to run the tests.
```bash
yarn test
```
or
```bash
npm test
```
